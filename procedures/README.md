## 📝 Structure d'une procédure

Chaque fiche suit une structure standardisée :

1. **📋 Contexte d'utilisation** — Quand et pourquoi utiliser cette technique
2. **🛠️ Prérequis** — Outils et conditions nécessaires
3. **⚙️ Procédure** — Étapes détaillées avec commandes
4. **📊 Résultats attendus** — Ce que l'on doit obtenir
5. **⚠️ Points de vigilance** — Aspects légaux et techniques
6. **🛡️ Contre-mesures** — Comment se protéger contre cette attaque
7. **📚 Ressources** — Liens pour approfondir
