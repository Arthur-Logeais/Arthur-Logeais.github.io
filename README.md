## 🎓 Certifications visées

Mon parcours de certifications est structuré par **paliers de difficulté progressive**, du pentesting fondamental à l'exploitation avancée :

| Niveau | Certification | Domaine | Statut |
|--------|---------------|---------|--------|
| 🟢 Intermédiaire | **CPTS** (HTB Certified Penetration Testing Specialist) | Pentesting infrastructure | 🎯 Objectif court terme |
| 🟠 Intermédiaire+ | **OSCP+** (Offensive Security Certified Professional) | Pentesting général + AD | 🎯 Objectif principal BTS |
| 🔴 Avancé | **CRTO** (Certified Red Team Operator) | Red Team avec Cobalt Strike | 🎯 Après OSCP+ |
| 🔴 Avancé+ | **CRTO II** (Certified Red Team Operator Level 2) | Red Team avancé, EDR evasion | 🎯 Spécialisation Red Team |
| 🟣 Expert | **OSCE3** (OSWE + OSEP + OSED) | Exploitation web avancée + Exploit dev | 🎯 Long terme |
| 🟣 Expert+ | **OSEE** (OffSec Exploitation Expert) | Kernel exploitation Windows | 🎯 Très long terme |

> 💡 Ce parcours représente plusieurs années d'apprentissage. Chaque certification validée fera l'objet d'un **retour d'expérience** détaillé dans ce portfolio.
